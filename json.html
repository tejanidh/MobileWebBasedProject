Getting Json DATA With Fetch & XMLHttpRequest (XHR).

  • Fetch API (fetch)
  • XMLHttpRequest (XHR)

JSON Concepts

JSON (JavaScript Object Notation) is a lightweight data interchange format that is easy for humans to read and write, and easy for machines to parse and generate.

It is widely used for data exchange between a server and a web application. JSON is compatible with many programming languages, including JavaScript.

JSON Examples

Example 1: Fetching JSON Data using Fetch API

Fetch JSON data from a local file using the fetch API. 

fetch('data.json')
  .then(response => response.json())
  .then(data => {
    console.log(data); // Handle JSON data here
    renderData(data); // Example function to render data
  })
  .catch(error => {
    console.error('Error fetching data:', error);
  });

function renderData(data) {
  // Example: Display data in HTML
  const container = document.getElementById('json-data');
  container.innerHTML = JSON.stringify(data, null, 2);
}

This code snippet fetches JSON data from a local file named `data.json` and processes it. It begins by using the `fetch` API to retrieve the data. Upon a successful response, the data is converted from JSON format using the `.json()` method. The processed data is then logged to the console and passed to a function named `renderData`. This function is responsible for rendering the JSON data into an HTML element with the ID `json-data`, displaying it in a formatted (pretty-printed) manner. If an error occurs during the fetch operation, it is caught and logged to the console.

Example 2: Fetching JSON Data using XMLHttpRequest

Use XMLHttpRequest to fetch JSON data asynchronously. 

var xhr = new XMLHttpRequest();
xhr.open('GET', 'data.json', true);
xhr.onreadystatechange = function() {
  if (xhr.readyState === XMLHttpRequest.DONE) {
    if (xhr.status === 200) {
      var data = JSON.parse(xhr.responseText);
      console.log(data); // Handle JSON data here
      renderData(data); // Example function to render data
    } else {
      console.error('Error fetching data:', xhr.statusText);
    }
  }
};
xhr.send();

function renderData(data) {
  // Example: Display data in HTML
  const container = document.getElementById('json-data');
  container.innerHTML = JSON.stringify(data, null, 2);
}

Example 3: Error Handling with Fetch API

Handle errors when fetching JSON data using the Fetch API. 

fetch('data.json')
  .then(response => {
    if (!response.ok) {
      throw new Error('Network response was not ok');
    }
    return response.json();
  })
  .then(data => {
    console.log(data); // Handle JSON data here
    renderData(data); // Example function to render data
  })
  .catch(error => {
    console.error('Error fetching data:', error.message);
  });

function renderData(data) {
  // Example: Display data in HTML
  const container = document.getElementById('json-data');
  container.innerHTML = JSON.stringify(data, null, 2);
}

This JavaScript code snippet fetches JSON data from a local file named `data.json` and includes comprehensive error handling. It uses the `fetch` API to request the data and checks if the response is successful using `response.ok`. If the response is not successful, it throws an error with an appropriate message. If the response is successful, it converts the response to JSON format using `response.json()`. The JSON data is then logged to the console and passed to a function named `renderData`, which displays the data in an HTML element with the ID `json-data` in a formatted (pretty-printed) manner. If any error occurs during the fetch process, it is caught and logged to the console with the error message.
