JavaScript Selectors

JavaScript selectors are used to select and manipulate HTML elements. Here are some common selectors:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
  • Fetch API (fetch)
  • XMLHttpRequest (XHR)

Examples

Example 1: getElementById()

Selects an element by its ID attribute and changes its text content. 

            document.getElementById('elementId').textContent = 'New Text';

Example 2: getElementsByClassName()

Selects all elements with a specific class name and applies a style. 

            const elements = document.getElementsByClassName('className');
            elements.forEach(element => {
              element.style.color = 'blue';
            });

Example 3: querySelector()

Selects the first element that matches a specified CSS selector and changes its text content. 

            document.querySelector('.example-title').textContent = 'Updated Example Title';

Example 4: querySelectorAll()

Selects all elements that match a specified CSS selector and applies a style to each. 

            const allExamples = document.querySelectorAll('.example');
            allExamples.forEach(example => {
              example.style.border = '1px solid #ccc';
            });

Example 5: Fetching JSON Data

Fetches JSON data from a local file using the fetch API. 

            fetch('data.json')
              .then(response => response.json())
              .then(data => {
                console.log(data); // Handle JSON data here
              })
              .catch(error => {
                console.error('Error fetching data:', error);
              });

Example 6: XMLHttpRequest to Fetch JSON Data

Uses XMLHttpRequest to fetch JSON data asynchronously. 

            var xhr = new XMLHttpRequest();
            xhr.open('GET', 'data.json', true);
            xhr.onreadystatechange = function() {
              if (xhr.readyState === XMLHttpRequest.DONE) {
                if (xhr.status === 200) {
                  var data = JSON.parse(xhr.responseText);
                  console.log(data); // Handle JSON data here
                } else {
                  console.error('Error fetching data:', xhr.statusText);
                }
              }
            };
            xhr.send();

About JavaScript Selectors and JSON

JavaScript selectors allow developers to interact with elements in the DOM dynamically. Incorporating JSON allows for efficient data exchange and manipulation within web applications.

These examples demonstrate how to use various selectors to access and modify HTML elements. Additionally, they showcase fetching JSON data both locally and from online sources using modern JavaScript techniques.
