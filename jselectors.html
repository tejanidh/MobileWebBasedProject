JSELECTORS

In javascript, we have different types of selectos such as by tag, by classname, by tag name and so on. Let's catch it up.

Select by Id

For this, we have one function called document.getElementById("ID_name"). This function is connected to whole html page. and whole HTML page is considered as a document. This function is used to select any tag by using id. So, it selects only one tag at a time.

Let's take an example. This section is linked with quick links section. if you will click it from there or else if you will select from this below button this section whole words case will be changed.

Example:

                    
    var t1 = document.getElementById("t1").innerHTML;
    t1 = t1.toLocaleUpperCase();
    document.getElementById("t1").innerHTML = t1;
    /*To make it to lower case*/
    var t1 = document.getElementById("t1").innerHTML;
    t1 = t1.toLocaleLowerCase();
    document.getElementById("t1").innerHTML = t1;

Select By Class name

Well By clicking the below button this sentence is going to wrapped by a light blue bg color document.getElementsByClassName("para")

So what this basically does is that it takes the paragraph by Class name and changes the sentence backgorung color since that is what the javascript file is coded to

Example:


                    function className(){
                      var p1 = document.getElementsByClassName("para");


                      for(x of p1){
                        x.style.backgroundColor = "lightblue";
                      }
                    }

So this code it takes the className of "para" paragraph and then by using for loop we are changing the color to backgorund color to light blue.
