Select by Id

For this, we have one function called document.getElementById("ID_name"). This function is connected to the whole HTML page, which is considered as a document. This function is used to select any tag by using an id. So, it selects only one tag at a time.

Let's take an example. This section is linked with the quick links section. if you will click it from this below button this section whole words case will be changed.

var t1 = document.getElementById("t1").innerHTML;

var t1 = document.getElementById("t1").innerHTML;
t1 = t1.toLocaleLowerCase();

Select By Class name

Well By clicking the below button this sentence is going to be wrapped by a light blue bg color. Syntax: document.getElementsByClassName(classname)

The getElementsByClassName(className) method in JavaScript is used to select all HTML elements that have a specific class name.

So what this basically does is that it takes the paragraph by Class name and changes the sentence background color since that is what the javascript file is coded to

How? My first p tag has class name "para" and the rest of p tags have different class names and that is why it only changes the color of the first paragraph

Example:

              
function className(){
  var p1 = document.getElementsByClassName("para");

  for(x of p1){
    x.style.backgroundColor = "lightblue";
  }
}

So this code takes the className of "para" paragraph and then by using a for loop we are changing the color to background color to light blue.

Select By Tag Name

To select elements by their tag name, we use the document.getElementsByTagName("tag_name") method. This method returns a collection of all elements with the specified tag name. Let's see how it works.

For example, let's say we want to change the text color of all paragraph elements. We can do this by selecting all `p` tags and applying the style change.

This is a paragraph.

This is another paragraph.

Yet another paragraph here.

Example:

              
function changeTextColor() {
  var paragraphs = document.getElementsByTagName("p");

  for (var i = 0; i < paragraphs.length; i++) {
    paragraphs[i].style.color = "red";
  }
}

document.getElementById("changeTextColor").onclick = changeTextColor;

In this code, we select all `p` elements and change their text color to red when the button is clicked.
